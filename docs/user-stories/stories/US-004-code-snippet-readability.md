# US-004 - Code Snippet Readability

## Epic

- EPIC-B Readable Learning Content

## Story

As a learner, I want readable code snippets in answers, so that I can understand examples quickly.

## Acceptance Criteria

1. Code uses syntax highlighting.
2. Code blocks support horizontal overflow without breaking layout.
3. Code remains readable on desktop and mobile.
