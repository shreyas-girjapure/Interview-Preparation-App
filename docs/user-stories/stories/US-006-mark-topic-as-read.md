# US-006 - Mark Topic as Read

## Epic

- EPIC-C Progress Tracking

## Story

As a signed-in learner, I want to mark a topic as read, so that I can track my conceptual progress.

## Acceptance Criteria

1. Topic page includes a read toggle.
2. Read state persists for the current user.
3. Read state is restored on refresh.
