# US-003 - Question-to-Topic Linking

## Epic

- EPIC-A Topic-First Discovery

## Story

As a learner, I want key topics in an answer to be clickable, so that I can follow a learning rabbit-hole.

## Acceptance Criteria

1. Question details show linked topics in answer content.
2. Clicking a linked topic opens the topic page.
3. Links are keyboard accessible and visually distinct.

## Status

- Implemented (Phase 2 baseline)
