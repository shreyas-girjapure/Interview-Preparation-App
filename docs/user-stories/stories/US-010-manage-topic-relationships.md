# US-010 - Manage Topic Relationships

## Epic

- EPIC-D Admin Content Management

## Story

As an admin/editor, I want to manage question-topic and topic-topic links, so that learning paths stay relevant.

## Acceptance Criteria

1. Admin can attach multiple topics to a question.
2. Admin can attach related topics to a topic.
3. Relationship changes are reflected on public pages.
