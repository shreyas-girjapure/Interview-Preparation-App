# US-001 - Topic Catalog

## Epic

- EPIC-A Topic-First Discovery

## Story

As a learner, I want to browse all topics, so that I can choose what to study first.

## Acceptance Criteria

1. Topic list is visible without sign-in.
2. Topics can be filtered or searched.
3. Each topic links to a dedicated topic page.
