# US-013 - Deduplication Controls

## Epic

- EPIC-E Content Operations v2

## Story

As a content maintainer, I want duplicate detection, so that we avoid noisy or repeated content.

## Acceptance Criteria

1. System flags possible duplicate slugs or near-duplicate titles.
2. Import does not auto-merge without user confirmation.
3. Review queue records dedupe decisions.
